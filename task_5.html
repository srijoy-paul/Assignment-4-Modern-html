Features of HTML5

  • Semantics
  • Multimedia Support
  • Canvas
  • Local Storage

Semantics

HTML5 introduces semantic elements like <header>, <footer>, <article>, providing a clearer structure for web pages.

Multimedia Support

Native support for embedding audio and video content using <audio> and <video> elements, eliminating the need for plugins.

Canvas

The <canvas> provides a drawing space for 2D graphics using JavaScript, facilitating interactive graphics and animations. Using this we can create graphics and animations directly within the browser.

Local Storage

It alows to store data locally on the user's device for more efficient and faster access.
